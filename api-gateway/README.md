# API网关服务

本目录用于存放API网关服务相关的配置和代码。

## 目录说明

该目录将包含以下内容：

- Kong/Tyk API网关配置文件
- API网关的Docker配置
- 路由规则和策略
- 插件和扩展
- 管理脚本和工具
